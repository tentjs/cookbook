Cookbook

This is the official cookbook for Tent. It's purpose is to showcase how Tent was intended to be used. As a "best practice" guide.

Each recipe will have the minimal styling required, and is written in Typescript which is not a requirement.

Examples

  • The classic To-do list
  • Pass data to component
  • Fetch data from API
  • Use session-/local storage

  • Accordion
  • Modal
